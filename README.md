## Rubik

Speedcubing stuff.

## To install

```bash
$ npm install
```

## Available Scripts

### `npm run dev`

Runs the app in development mode.
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
The page will reload if you make edits.

### `npm run storybook`

Run the storybook.

### `npm run build`

Builds the app for production to the `dist` folder.
